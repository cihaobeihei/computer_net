# expr函数的使用

## 1.expr间接
expr函数本来最多是用来计算整数的，但我们可以通过一些手段使expr函数功能更丰富一些。<br>

## 1.判断一个数是否为整数
```shell
#!/bin/bash
expr 2 + $1 &>/dev/null
if [ $? -eq 0 ]
then
        echo "$1 is int"
else
        echo "$1 is not int"
fi
```

## 2.判断一个文件的扩展名是否为.txt
```
#!/bin/bash
expr "$1" : ".*\.txt$" &>/dev/null
if [ $? -eq 0 ]
then
        echo "$1 is regular file."
else
        echo "$1 isn't regular file."
fi
```

## 3.计算字符串长度
```
expr length "$noble"
```
