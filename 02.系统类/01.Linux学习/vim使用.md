# vim使用
## 01.分屏操作
- ``:sp``上下分屏
- ``:vsp``左右分屏 后面加文件名可以打开另外的文件6
- ``Ctrl+ww``切屏
- ``:wq``推出当前屏



## 02.常用命令

```shell
h,j,k,l         #移动
0               #移动行首
$               #移动到行尾
+               #移动光标到下行行首
-               #移动光标到上行行首
gg              #移动光标到文件首字符
shift+g         #移动光标到文件结尾
w               #移动光标到下个单词 词首
e               #移动光标到下个单纯 词尾
b               #移动光标到上一个单词

y               #复制选中字符
nyy             #复制n行
p               #粘贴

nx              #删除当前光标以及右方n个字符
shift+c         #删除当前光标到行尾，并进入insert模式
ndd             #剪切n行(可以用p粘贴)
```
