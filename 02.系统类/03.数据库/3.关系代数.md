# 关系代数
关系代数是抽象的关系表达查询，插入，删除等语句。是理论研究，实际价值不大。<br>
这里做简要概述。<br>

## 1.运算符
### 1.传统集合运算符
包括并、查、交、笛卡尔积。<br>
只介绍笛卡尔积的运算规律。<br>

R:<br>

| A | B | C |
|--|---|  --|
| a1 | b1 | c1 |
| a1  | b2  | c2  |
| a2   | b2  | c1  |

S:<br>

| A | B | C |
|---|---|---|
| a1   | b2   | c2   |
| a1   | b3   | c2   |
| a2   | b2   | c1  |


R*S:<br>

| R.A | R.B | R.C | S.A | S.B | S.C |
|-----|-----|-----|-----|-----|-----|
| a1 | b1 | c1 | a1   | b2   | c2   |
| a1 | b1 | c1 | a1   | b3   | c2   |
| a1 | b1 | c1 | a2   | b2   | c1  |
| a1  | b2  | c2  | a1   | b2   | c2   |
| a1  | b2  | c2  | a1   | b3   | c2   |
| a1  | b2  | c2  | a2   | b2   | c1  |
| a2   | b2  | c1  | a1   | b2   | c2   |
| a2   | b2  | c1  | a1   | b3   | c2   |
| a2   | b2  | c1  | a2   | b2   | c1  |

从上面的实例中我们可以看到R表中的每个元组分别和S中的三个元组联合。<br>
