## 2.StringBuilder

### 1.构造方法

```java
StringBuilder(String s);         //构造一个字符串生成器，并初始化为指定的字符串内容
```


## 4.Object

```java
String toString();
```

是java的根类，所有的类都默认继承Object类<br>

### 1.toString方法
toString方法有以下几个特征：<br>
- 1.对于一个实例化对象，会默认调用该方法如：
  ```java
  Student s = new Student();
  System.out.println(s.toString());
  System.out.println(s);//和上面的语句作用相同
  ```
- 2.我们通常会重写此类，让该类的输出更有价值。
  ```java
  public String toString() {
    return name + "@" + age;
  }
  ```

### 2.equals方法
```java
boolean equals(Object obj);
```
使用==来比较两个对象是否相等，默认比较地址值是否相等。<br>
一般我们也会重写该方法，让判断两个类是否相等的标准更合理，比如当成员变量都相等时，表示两个类相等。<br>

例如：<br>
```java
@Override
Public boolean equals(Object o) {
  //compale address
  if(this == o) {
    return true;
  }

  //improve stability
  if(this.getClass() != o.getClass()) {
    return false;
  }

  Person other = (Person) o;
  if(!this.name.equals(other.name)) {
    return false;
  }
  if(this.age != other.age) {
    return false;
  }

  return true;
}
```

## 5.System
System不可实例化<br>

### 1.arraycopy
```java
static void arraycopy(Object src, int srcPos, Object dest, int destPos, int length);
```

参数说明：<br>
- src:原数组
- srcPos:原数组的起始索引位置
- dest:目标数组
- destPos:目标数组的索引起始位置
- length:指定接受的元素个数

### 2.currentTimeMillis方法
```java
static long currentTimeMillis()
```
以毫秒值返回当前时间戳(1970-1-1 00:00:00)。<br>

### 3.exit方法
```java
static void exit(int status);
```
终止当前正在运行的Java虚拟机。参数用作状态码；非0状态码一般表示异常终止。<br>

### 4.gc方法
```java
static void gc();
```
Java中的垃圾回收器，JVM虚拟机会自动调用。<br>

## 8.Calendar类
提供了一些操作年月时的静态方法。<br>

### 1.getInstance方法
```java
static Calendar getInstance();
```

实例1(获取时间)：<br>
```java
Calendar c = Calendar.getInstance();

//void get(int field);
int year = c.get(Calendar.YEAR);
int month = c.get(Calendar.MONTH) + 1;
int day = c.get(Calendar.DAY_OF_MONTH);

System.out.println(year + "-" + month + "-" + day);
```

实例2(修改时间)：<br>
```java
void set(int field, int value);
c.set(Calendar.DAY_OF_MONTH, 20);

void add(int field, int amount);//在指定时间上加上amount的值
c.add(Calendar.DAY_OF_MONTH,1);
```
