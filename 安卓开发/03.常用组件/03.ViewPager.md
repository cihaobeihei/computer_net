# ViewPager
ViewPager可以实现页面左右滑动,常见的应用场景如(应用启动时的功能介绍|相册的图片预览|App导航)<br>

## 1.实现ViewPager简单方法

### 1.1.里程碑一 创建一个Views集合
```java
private int[] mLayoutIDs = {
    R.layout.view_first,
    R.layout.view_second,
    R.layout.view_third
};
private List<View> mviews;
for (int i = 0; i < this.mLayoutIDs.length; i++) {
    mviews.add(getLayoutInflater().inflate(mLayoutIDs[i], null));
}
```

### 1.2.里程碑二 实现PagerAdapter成员变量
```java
PagerAdapter mPagerAdapter = new PagerAdapter() {
    @Override
    public int getCount() {        //返回有几个pager
        return mLayoutIDs.length;
    }

    @Override
    public boolean isViewFromObject(@NonNull View view, @NonNull Object o) {  //判断instantiateItem返回的Object是否和View是同一个对象
        return o == view;
    }

    @NonNull
    @Override
    public Object instantiateItem(@NonNull ViewGroup container, int position) { //将当前position的View加入container并返回
        View child = mviews.get(position);
        container.addView(child);
        return child;
    }

    @Override
    public void destroyItem(@NonNull ViewGroup container, int position, @NonNull Object object) { //将当前position的View移除container
        container.removeView(mviews.get(position));
    }
};
```

### 1.3.里程碑三 添加PagerAdapter到布局文件ViewPager中
```xml
<!-- 创建ViewPager布局文件 -->
<?xml version="1.0" encoding="utf-8"?>
<LinearLayout
    xmlns:android="http://schemas.android.com/apk/res/android" android:layout_width="match_parent"
    android:layout_height="match_parent">

    <!--v4扩展包 提供的ViewPager -->
    <android.support.v4.view.ViewPager        
        android:id="@+id/view_pager"
        android:layout_width="match_parent"
        android:layout_height="match_parent">
    </android.support.v4.view.ViewPager>
</LinearLayout>
```

```java
ViewPager mviewPager = findViewById(R.id.view_pager);
mviewPager.setAdapter(mPagerAdapter);
```
