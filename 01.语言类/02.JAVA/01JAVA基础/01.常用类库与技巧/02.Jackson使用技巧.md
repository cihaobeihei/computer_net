# Jackson使用技巧

## 1.DIY一个类的返回格式

```java
@JsonInclude(Include.NON_NULL) //可以放在类上，也可以放在字段上
public void User{
    private String name;
    
    @JsonIgnore         //忽略该字段
    private String age;
    
    @JsonFormat(pattern="yyyy-mm-dd hh:mm:ss a", locale="zh", timezone="GMT+8") //格式化
    private Date birth;
    
    @JsonInclude(Include.NON_NULL) //设置显示模式(当desc不为null才显示)
    private String desc;
}
```

上面代码显示了三种Jackson三种常用的注释(作用写在后面的注释中)。
