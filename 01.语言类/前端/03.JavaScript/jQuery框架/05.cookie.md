# cookie使用

在前端编程时想要使用context,可以使用cookie.<br>

## 1.引入
使用和jquery配套的cookie框架:<br>
在当前路径下``jquery.cookie.js``<br>
```js
<script src="js/jquery.cookie.js"/>
```

## 2.添加cookie,读取cookie和删除cookie
```js
$.cookie("key","value");    //添加cookie
$.cookie("key");            //读取cookie
$.cookie("key", null);      //删除cookie
```
