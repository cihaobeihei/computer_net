# 最常使用标签及属性

## 1.表格
实现一个表格的最基本标签是：<br>
```html
<table>
  <tr>
    <tb>contont1</tb>
    <tb>contont2</tb>
  </tr>
</table>
```

``<tr>``标签标示行，``<tb>``标签标示基本的一个表格单元。<br>

<table>
  <tr>
    <tb>contont1</tb>
    <tb>contont2</tb>
  </tr>
</table>

想要使用表格表达更多类型的信息，通常会使用到表格的合并与拆分。<br>
