### 2.2.编写代码
```html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8" />
		<title></title>
		<script type="text/javascript" src="js/jquery-1.11.3.min.js"></script>
	</head>
	<body>
		
	</body>
	<script>
		$(document).ready(function(){
			alert("Hello World!");
		});
	</script>
</html>
```

当页面加载完毕后就会执行函数<br>
